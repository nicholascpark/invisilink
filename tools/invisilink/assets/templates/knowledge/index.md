# Knowledge Index

*Master index of all wiki pages. Maintained by the librarian. One bullet per page, grouped by type. Within each group, entries are sorted by `updated` date descending.*

> **Entry format:** `- [[page-id]] — one-sentence summary (max 200 chars). (confidence, evidence-count-if-applicable, updated YYYY-MM-DD)`
>
> Pattern pages cite evidence count (e.g., `(cross-venture-pattern, 2 ventures, updated 2026-04-13)`). All other types cite confidence level only (e.g., `(high, updated 2026-04-13)`). See `schema.md` for the full specification.
